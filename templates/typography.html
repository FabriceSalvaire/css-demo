Introduction

There is 3 principal types of fonts:

  • Sans-Serif typeface like the one used in this text.
  • Serif typeface which have a small line attached to the end of a stroke in a glyph.

    It forms a virtual line which help to read the text of a paragraph.
    It is well adapted for printed book but it can be less suitable on electronic device where the dpi resolution is limited.

  • Monospaced typeface where all glyphs have the same width, usually a e is shorter than a m.

    This typeface is used when an homogeneous character spacing is required, for example to show a source code.

Additionally, it exists a plenty of fancy fonts, like curvy etc.

Rule: A font must be readable on all devices by everybody.

The main factors that affect the readability of a font are:

  • font design
  • font size
  • font weight (stroke width)
  • contrast between the type colour and the background
And these factors which cannot be chosen by the author:
  • media resolution (dpi)

    Which have a large range nowadays.

  • anti-aliasing algorithm applied to the font by the raster engine

    It can be observed with a magnifier tools or using a screenshot.

Rule: We have to choose a font, a size (weight) and a contrast which work together on all devices (responsive design).

Rule: The relative importance of the different parts of the text must be established by a consistent couple of font styles.

Accessibility Rule: Peoples with visual impairment need a larger font size. It is mandatory on wide audience site.
